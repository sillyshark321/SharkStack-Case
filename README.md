## Materials List
| Material | Count | 
| --- | --- |
| 3mm or 1/8" thick, 24" x 12" sheet of acrylic | 1 |
| M2 x 12 mm standoffs* | 14 (18 with OLED covers) |
| M2 x 8 mm screws | 22 (28 without tenting) |
| M2 x 18 mm screws** | 6 |

(* I used 11 mm tall standoffs, works just fine)\
(** Only for tenting, adjust screw lengths if you wish to add more tenting layers)

## Assembly
